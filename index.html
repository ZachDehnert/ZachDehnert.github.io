Scratch Program One

Create an account at http://scratch.mit.edu if necessary.

Complete the following Scratch problem:

1. Put a small sprite of your choice into each corner of the stage.

2. Using the "friction marble" sprite have the user touch all four sprites in the corners. Have the program stop when the last sprite is touched.

- Friction Marble Sprite Download

- Optionally display the number of seconds it took to "beat" the game.
