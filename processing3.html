From the p5 examples I chose an input program that takes any input from the keyboard that is a letter. If you hit a non letter key it will reset so you can create a new image. I modified the canvas size when I typed it in. Then I modified the r, g and b variables to only display pastel colors instead of the whole color range of (0-255). The last modification I made was the rectangle width. It was set to be a quarter of the size of the canvas, and I made it wider, by being roughly have the size of the canvas. The program takes input from the user and it generates a random number for each variable for r, g, and b colors. It then takes those values (0-100) and it concatenates those value so that on each input from the user it is a random color. Pretty cool! Here is the original keyboard example
